## 关闭防火墙命令
关闭公网上的防火墙
`netsh advfirewall set  currentprofile state off`

关闭内网中的防火墙
`netsh advfirewall set privateprofile state off`

### 远程关闭防火墙
```
netsh -r 192.168.123.45 -u administrator -p 123456 -c advfirewall set allprofiles state on

```
